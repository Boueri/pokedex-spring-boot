# 🚀 Pokédex Desktop - Java + Spring Boot + JavaFX

![Spring Boot](https://img.shields.io/badge/Spring_Boot-3.4.1-green)

Uma Pokédex desktop desenvolvida em **Java**, utilizando **Spring Boot**, **JavaFX** e a **PokéAPI**. O projeto foi criado com foco em boas práticas de desenvolvimento, arquitetura organizada e interface moderna, simulando uma aplicação desktop profissional.

# ▶️ Como executar

## Clonar

```bash
git clone https://github.com/Boueri/pokedex-spring-boot.git
```

## Entrar na pasta

```bash
cd pokedex-spring-boot
```

## Executar

```bash
.\mvnw javafx:run
```

---

# 📦 Gerar Executável

Após gerar o pacote:

```bash
jpackage
```

A aplicação pode ser distribuída como:

```
Pokedex.exe
```
